How HTML access CSS.

HTML and CSS are the two main part that make up a website. HTML makes the base website, it's like the foundation of a house and then CSS is the part that makes it look good so like carpet or the paint on the walls. Here are three ways they link.


Inline is the easiest one to use if you are making a single page website. Inline is adding a style attribute to one element instead of changing the whole look of the page. Inline will have to be implemented to each element instead of controlling all elements from one CSS file


Internal CSS is when the style attribute is added into the head element instead of being in the same line as the header or paragraph that you have made. Every Time you want to add a new style to a part of code you will need to add a new style instead of putting it in a new head element.


External CSS is when you create a new file in a folder (make sure you know where it is) you can call the file mystyle.css but make sure it is a .css file. All you have to do then is create all the lines of code for designs.

Box models for CSS.

Box models are the layout of the website which allows you to space things out add text in the center, left or right. You can add padding to make things stand out and add a border between different things. And you can add a margin to stop text and photos from going to the edge of the website.


The content box is in the centre of the page between the margin, border and padding. The content is the stuff like text, photos and graphs.


Padding is placed around the content to create and invisible barrier to stop text and photos colliding.


A border is placed around the padding which is around the content. The border adds a solid color so that it looks different. The border can just be a base color the same as the background or be a different color to make it look nicer.


The margin is a bit of code that clears a space around the area of the website. The margin is a clear area so makes no difference to the background.

Fundamentals Of a Scripting Language.

Scripting language is a programming language that supports scripts like run time programs. On this i will be explaining the fundamentals of scripting language.


Variables are something that will always change, depending on the program used and what software it is used for can be a very good add-on. A variable is something that holds a value an example of this is:
var price1 = 5;
var price2 = 6;
var total = price1 + price2;


Loops are continuous. In programming they are what they are called. They will repeat the process until there is a certain value met or the limit is reached.


Functions are used to create a sequence that a person has to carry out and to get a end result. Each button click is a function because you will get a result. An operating system is also a function because when click the start button or right click you get a result because the function has been completed.
